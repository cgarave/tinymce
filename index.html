Guide for new users

1. Promotion Automation - this is where you create and edit your T&C promotions.

2. HTML Filename - file name of your T&C promotion. (ex. MY_SPPT_11_Sep_2024_005)

    3. Import - this is where you import your existing T&C promotion if you want to edit or add something to the file.
  1. Open the HTML file using an application that can open HTML files (e.g., Visual Studio Code, Adobe Dreamweaver, Notepad++) and copy all the codes.
  2. After pasting all the code into the import box, the text will appear in the editor.
    4. Uncheck if not imported
  1. Checkbox will remain unchecked if you don't import codes.
  2. The checkbox will automatically be checked when you import a code. However, if you delete all the contents or want to create T&C from scratch, you will need to uncheck the checkbox if it is checked.

5. Select templates - templates are designed to allow users to easily create and edit their T&C promotions without needing any knowledge of editing HTML files.

    6. Select region - select the region for which you want to create T&C.
  1. After selecting region, a generated text will appear in the editor.
  2. Note: Only edit the text under Write/Paste Significant/Full Promotion Contents here. Do not delete titles such as Significant Conditions and Full Promotion Specific Terms and Conditions.

7. Preview - a preview of your T&C promotion when deployed to production.

8. Download - a filename must be provided before clicking the download button.

Current limitations and its workaround

  1. Issue: Copying numbered text with lower-roman or lower-alpha formatting from Microsoft Word will not work.
  2. Workaround: Convert all lower-roman and lower-alpha formats to numbers in Microsoft Word before pasting into the editor, then use the automation's editor to convert the numbers back to lower-roman or lower-alpha.
  1. Issue: Copying numbered text without a dot (e.g., ❌ 1 instead of ✅ 1.) from Microsoft Word will not work.
  2. Workaround: Convert the numbers with the right format on Microsoft Word before pasting to get the right result. (e.g., ✅ 1.)
  1. Limitation: Copying of images directly to the automation's editor.
  2. Workaround: Message the Web team (your region's POC) to deploy the image to the server. Once the image link is provided, paste the link to editor and it will automatically convert to image.
