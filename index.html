Guide for new users

1. T&C Automation - this is where you create and edit your T&C promotions.

2. HTML Filename - file name of your T&C promotion. (ex. MY_SPPT_11_Sep_2024_005)

    3. Select region - select the region for which you want to create T&C.
  1. After selecting region, a generated text will appear in the editor.
  2. Note: Only edit the text under Write/Paste Significant/Full Promotion Contents here. Do not delete titles such as Significant Conditions and Full Promotion Specific Terms and Conditions.

4. Preview - a preview of your T&C promotion when deployed to production.

5. Download - a filename must be provided before clicking the download button.

Current limitations and their workarounds

  1. Issue: Copying numbered text with lower-roman or lower-alpha formatting (e.g., a b c/i ii iii) from Microsoft Word will not work.
  2. Workaround: Convert all lower-roman and lower-alpha formats to numbers in Microsoft Word before pasting into the editor, then use the automation's editor to convert the numbers back to lower-roman or lower-alpha.
  1. Issue: Copying numbered text without a dot (e.g., 1 2 3) from Microsoft Word will not work.
  2. Workaround: Convert the numbers with the right format on Microsoft Word before pasting to get the right result. (e.g., ✅ 1. 2. 3.)
  1. Limitation: Copying of images directly to the automation's editor.
  2. Workaround: Message the Web team (your region's POC) to deploy the image to the server. Wait for the image link provided by Web team, paste the link to editor and it will automatically convert to image.
